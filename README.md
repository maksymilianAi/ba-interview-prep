# Interview Prep — Product Owner / BA / AI PM

Статичний сайт для підготовки до двох BA/product-співбесід. Три сторінки, мобільна верстка, без залежностей і збірки.

## Сторінки

- **[index.html](index.html)** — Основи PO / PM: коротка теорія по ключових темах + питання для самоперевірки + тест із варіантами відповідей.
- **[vacancy-legal-po.html](vacancy-legal-po.html)** — Вакансія 1: Product Owner / BA (юридичні рейтинги, London) — розбір домену, ключові навички під вимоги, питання.
- **[vacancy-ai-pm.html](vacancy-ai-pm.html)** — Вакансія 2: AI Product Manager (BABOK-BA + продукт + AI) — техніки BA, BPMN/UML, вимоги, AI-в-продукті, питання.

## Запустити локально

Відкрити `index.html` у браузері або підняти сервер:

```bash
python3 -m http.server 8000
# http://localhost:8000
```

## Опублікувати через GitHub Pages

Settings → Pages → Source: Deploy from a branch → Branch `main`, папка `/ (root)` → Save.
Сайт буде за адресою `https://<нік>.github.io/<репо>/`.
